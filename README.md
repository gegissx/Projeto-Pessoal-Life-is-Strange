# Projeto Pessoal - Life is Strange
Desenvolvido por: Geovanna Cristina Paschoal.


## Descrição
Este projeto tem como objetivo o desenvolvimento de um blog inspirado no jogo Life is Strange, com foco na reflexão sobre decisões, emoções e suas consequências, incentivando a escrita como ferramenta para refletir sobre vivências pessoais.
---

## Objetivo
Desenvolver uma plataforma semelhante a um blog que permita o registro e a reflexão sobre experiências pessoais, possibilitando definir sentimentos presentes em diferentes situações e consequentemente, promover o desenvolvimento pessoal do usuário ao auxiliar na compreensão de suas emoções.
---

## Funcionalidades
- Página home;
- Cadastro e Login;
- Publicação de posts;
- Interação entre usuários através de comentários;
- Dashbard com resumo de interações e sentimentos mensais;
